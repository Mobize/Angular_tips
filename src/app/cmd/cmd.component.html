Commandes via CLI

Mise à jour d'Angular

npm install -g @angular/cli@latest

Générer un composant

ng generate c <nom du composant>
Exemple: ng generate c login créera login.component.ts (Class LoginComponent)

Générer un service

ng generate s <nom du service>
Exemple: ng generate s user créera user.service.ts

Lancement du serveur local et ouverture dans le navigateur

ng serve -o
